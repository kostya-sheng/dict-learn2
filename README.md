# Dict Learn 2

A basic static English-Chinese bilateral word lookup app.

## Features

- English to Chinese lookup
- Chinese to English lookup
- Pinyin search support
- Example sentences in both languages
- Browser-local study list

## GitHub Pages

This project is designed to run directly from GitHub Pages with no build step.

Recommended Pages settings:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/ (root)`
